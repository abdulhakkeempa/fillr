The friction you know too well

You're reading documentation. You copy a command. You paste it into your terminal. Then you manually hunt for every and replace it. Hope you didn't miss one.

Fillr makes placeholders editable

Click on any placeholder to edit it inline. Copy the final command with one click. No more manual find-and-replace.

How it works

1

Install extension

Add Fillr to Chrome from the Web Store or install from source.

2

Visit documentation

Browse any docs site. Fillr automatically detects code blocks with placeholders.

3

Edit and copy

Click placeholders to edit them. Hit the copy button to get your final command.
